¿Qué es Chile Hero?


Este es un proyecto de canciones chilenas principalmente para el Guitar Hero World Tour

¿Porque se inició este proyecto?


Resulta que muchos jugadores chilenos que juegan Guitar Hero, querian tener canciones chilenas, pero resulta que las canciones que estaban para el juego,
estaban mal hechas, por lo tanto como yo ya tengo experiencia creando canciones para el guitar hero, tomé la iniciativa de crear canciones con artistas chilenos

¿Para que plataforma será compatible?


Como mencioné anteriormente, el proyecto será principlamente para el Guitar Hero World Tour, sin embargo esto también seran compatibles con Clone Hero, Yarg, Fret Smasher

¿Esto será para cualquier tipo de jugadores?


Si, para los jugadores que se esten iniciando con Guitar Hero, no tendrán problemas ya que tendrán todas sus dificultades, sin embargo no lo reduciré de forma manual,
por cosas de tiempo, yo lo hago con un programa que se llama "Onyx Music Game Toolkit"

¿Los charts tendran solo guitarra?


No, tendrán guitarra y bajo, ahora si deseas ayudarme, que más abajo lo indico, y sabes hacer bateria y vocals, bienvenido
